# Tombinoclock

## Installation du projet

```bash
# Installation des dépendances
npm install

# Copie du fichier .env
cp .env.example .env
# N'oubliez pas de modifier les variables d'environnement dans le fichier .env si besoin
```

## Lancement du serveur

```bash
npm run dev
```
