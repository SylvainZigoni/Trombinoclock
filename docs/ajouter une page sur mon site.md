# Ajouter une page sur mon site

## Étapes

1. On créer un controller qui va réaliser l'action. Ici d'afficher la page.
2. On créer une route qui va appeler ce controller.
3. On créer la vue qui va être affichée.
4. On ajoute un lien dans la navigation pour accéder à cette page.
5. On oublie pas de récupérer les données dans le controller si besoin.
